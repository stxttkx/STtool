# 抓盘

## 介绍

抓盘是一款用于获取网盘资源分享的工具，旨在帮助用户免受广告和登录的烦恼。 

> 注意：该应用无需安装！任何要求你先安装后使用该应用的个人或组织均是非官方，提供的软件大多含有病毒，因使用非官方软件而造成的损失请自行承担。

## 软件架构

该应用为HTML网页。

## 使用说明

1. 将仓库Clone到本地
2. 打开此文件夹中的`index.html`
> 确保你的浏览器可以访问本地的`index.js`。
